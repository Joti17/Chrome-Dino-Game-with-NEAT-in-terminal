# Chrome-Dino-Game-with-NEAT-in-terminal
An implementation of the chrome Dino game fully made in the terminal and python


# How to use
```bash
git clone https://github.com/Joti17/Chrome-Dino-Game-with-NEAT-in-terminal.git
cd Chrome-Dino-Game-with-NEAT-in-terminal
pip install -r requirements.txt
python main.py
```
